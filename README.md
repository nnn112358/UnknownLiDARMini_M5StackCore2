# UnknownLiDARMini_M5StackCore2

https://github.com/user-attachments/assets/456000e1-3c88-458d-b501-94e88ccd649b
操作：
Aボタン：縮小
Bボタン：ポイント表示、ポリゴン表示切替
Cボタン：拡大

謎LiDAR を M5StackCore2 (ESP32) , PlatformIO で使うコードです。
